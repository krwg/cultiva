Notifications

Quiet, timely reminders. Cultiva uses the system notification center — the same channel as Mail and Calendar on Apple platforms.

💻
Desktop app only

Install and run Cultiva for Windows or macOS to receive native reminders. The browser preview cannot show system notifications.

General
Allow notifications Master switch for habit and calendar reminders.
Habits
Daily summary Once per day if you still have habits to complete.
Time Uses the timezone set under Calendar.
Calendar
Event alerts Heads-up before today’s events from your Cultiva calendar.
Advance notice How early to notify before the event starts.

If alerts do not appear, check Cultiva in System Settings → Notifications (iOS, iPadOS, macOS) or Windows Settings → Notifications.
