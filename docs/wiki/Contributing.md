# Contributing

See the canonical **[CONTRIBUTING.md](https://github.com/krwg/cultiva/blob/main/CONTRIBUTING.md)** in the repository.

## Quick summary

1. Fork → branch from `main`
2. `npm ci && npm run lint && npm test && npm run build`
3. English commit messages and PR descriptions
4. i18n: add **en + ru** keys in `src/core/i18n.js` for user-facing text
5. Plugin/registry changes → coordinate with [cultiva-plugins](https://github.com/krwg/cultiva-plugins)

## Where to contribute

| Change | Repo |
|--------|------|
| Cultiva app | krwg/cultiva |
| Official plugins | krwg/cultiva-plugins |
| Wiki pages | `docs/wiki/` → publish to GitHub Wiki |
