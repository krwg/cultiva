# Cultiva

**Grow habits. Grow yourself.**

Cultiva is an offline-first desktop habit tracker. Each habit grows from a **seed** to a **legacy tree** on your machine — no cloud, no account, no telemetry.

**Current release:** [1.7.0 · Linden](https://github.com/krwg/cultiva/releases/tag/1.7.0)

## How it works

```
Seed → Sprout → Sapling → Tree → Legacy (365+ days)
```

Plant habits in your **garden**. Mark them complete. Watch plants evolve. Open the **calendar** for events alongside habits.
